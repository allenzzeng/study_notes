### 1. 下载项目

使用 [DownGit](https://downgit.github.io/) 下载项目：
1. 访问 https://downgit.github.io/
2. 粘贴项目URL（如：` https://github.com/allenzzeng/study-notes/edit/main/01 `）
3. 点击下载按钮获取项目文件
